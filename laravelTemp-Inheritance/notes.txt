
 ** Blade Template:Inheritance **
 **Blade Template:Inheritance is a very powerful feature of Blade templating engine.Using Blade Template:Inheritance we can create a parent template and extend it as many times as we need to. The parent template is defined with the `@section` and `@show` directives. The child template which extends the parent template is defined with the `@extends` directive.**
