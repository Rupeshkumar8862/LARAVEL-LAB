** Pass the value inside Contrller with the help of Route
**in route
Route::get('/user/{id}', [PageController::class, 'showUser'])->name('user');
**in Controller
public function showUser(string $id){
        // return "<h1>Welcome to Controller</h1>";
        return view('user',['id'=>$id]);
        *or use compact for same name key and varable
        return view('user',compact('id'));
    }
